About NUS Module Mappings

Every year, many NUS students apply for the Student Exchange Programme. However, the process of application is complicated and the information is presented in a PDF file making it is extremely tedious and takes a long time to search for the relevant modules that students want to take at their partner universities.


We aim to create a web application with a database of module mappings to make the searching process for possible module mappings faster and easier via different filtering criteria, and to be able to track and store a list of modules (with their mappings) that the student intends to take on exchange.
